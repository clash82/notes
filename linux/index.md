## [tmux] attach to existing sessions: ##

`tmux attach`

## [tmux] keyboard shortcuts: ##

`Ctrl+B + C = new session
Ctrl+B + N = next session
Ctrl+B + W = session list
Ctrl-B + 1..9 = switch to specified session
Ctrl-D = kill current session`

## Clear directory: ##

`rm -rf %path%`
